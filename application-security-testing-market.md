<p><ul><li>Static AST</li><li>Dynamic AST</li><li>Interactive AST</li><li>Mobile AST</li></ul></p>
<p><p>Application Security Testing (AST) can be divided into four main market types: Static AST, Dynamic AST, Interactive AST, and Mobile AST. Static AST involves scanning an application's source code to identify potential security vulnerabilities. Dynamic AST involves testing the application in a live environment to uncover vulnerabilities during runtime. Interactive AST combines static and dynamic testing to provide comprehensive security assessment. Lastly, Mobile AST focuses specifically on mobile applications, testing their security on various platforms and devices. These different types of AST help organizations ensure the security of their applications and protect against potential cyber threats.</p></p>
